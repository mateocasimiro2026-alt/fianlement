Casimiro Photography Website
---------------------------------
1. Place your white logo inside the "images" folder and name it "logo.png".
2. Place your candid photo inside the "images" folder and name it "candid.jpg".
3. Edit index.html if you want to change text or package prices.
4. To deploy on Vercel:
   - Go to https://vercel.com/
   - Create an account (or log in).
   - Create a new project and import this folder.
   - Connect your domain (casimirophotography.com).
5. Later, update the package buttons with your Stripe payment links.

That's it! Your site will be live.
